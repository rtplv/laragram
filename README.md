## About Laragram

Yet another pet-project with Instagram-like app implementation. 

## Stack

- PHP 7.2
- Laravel 7.2
- Vue 2.5

## Startup guide

1. Install dependencies

```bash
composer install
npm install
```

2. Generate app key
```bash
php artisan key:generate
```

3. Build frontend

- for development:  `npm run hot`
- for production: `npm run prod`
